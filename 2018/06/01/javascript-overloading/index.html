Javascript 的重载

什么是重载

​ 所谓重载就是一组具有相同名字，不同参数个数的函数，可以根据传入的不同参数个数，来决定使用不同的函数。

​ 理论上来说，Javascript是没有重载的，因为后定义的函数会覆盖前面的同名函数。

​ 如果一定要达到重载的方法，看下面的叙述。
